# UH Maui College Course Catalog

## Setup Instructions

1. Open the `basic-json` folder in VS Code or your preferred editor.
2. Open `index.html` in your browser, or use a local server (e.g., Live Server extension) for full functionality.
3. All features work client-side; no Node.js server is required for this version.

## Features
- Robust JSON parsing and validation
- Multi-field search and combined filters
- Responsive, accessible, blue-themed UI
- Rich course cards, statistics, and modal details
- Export and add course functionality

## Testing
- Handles valid, invalid, and empty JSON data
- Search and filter combinations supported
- Responsive on mobile and desktop
- Performance tested with large datasets
