# HW12-A: Hawaii Island Cards

This React app implements the Week 12 React Basics assignment by rendering three Hawaii island cards (Maui, Oahu, and Kauai) using JSX, functional components, props, ES6 imports/exports, and `.map()` over a local data array in App.jsx; IslandCard is defined in its own component file, imported into App.jsx, and styled with CSS so the final output is readable and organized. AI tool use attribution: GitHub Copilot (GPT-5.3-Codex) was used to help scaffold files and draft component structure, and all final content and requirements were reviewed before submission.

## Feature Checklist (Rubric)

- [x] JSX used for all UI (no `React.createElement` calls)
- [x] Functional components used (`App`, `IslandCard`, optional `Header`)
- [x] Props passed to `IslandCard` (`name`, `description`, `tip`)
- [x] ES6 modules used (`export default` / `import`)

- [x] Basic styling applied with CSS (`App.css`, `index.css`)
